Feature Extraction Library - FELib

  FELib Version 1.2, updated on Dec. 31, 2008. Before downloading the material, please read and agree to the terms of using our datasets and/or implementation.
 

Readme

  • This software extracts the five kinds of features: 1. Color histogram, color moments. 2. Edge histogram. 3. Gabor wavelets transform. 4. Local Binary Pattern. 5. GIST.

  • Requirement: Matlab 7.3 or above, OpenCV library. For Mac OSX, you need install OpenCV according to this guide.

  • Install and usage. Please follow the instruction described in readme.

  • Download code.

  • Bug reports are welcome. Please write to Jianke Zhu (jianke.zhu_at_gmail.com).
